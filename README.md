# Sistema de Gestão de Produção

Projeto em desenvolvimento para automatizar o cálculo de custos de materiais e auxiliar na precificação de produtos.

## 🎯 Objetivo

Criar um sistema simples para registrar materiais utilizados na produção, calcular custos e facilitar a tomada de decisões sobre preços de venda.

A ideia é evoluir o projeto gradualmente, adicionando funcionalidades comuns em sistemas de gestão de produção.

## 🧰 Tecnologias Utilizadas

- Python 3.11
- Flask
- PostgreSQL
- SQLAlchemy
- HTML
- JavaScript (Fetch API)

## ⚙️ Funcionalidades Implementadas

Atualmente o sistema permite:

- Cadastro de materiais
- Listagem de materiais cadastrados
- Exclusão de materiais
- Edição inline diretamente na tabela
- Comunicação assíncrona com o backend usando `fetch()`

A edição inline permite alterar os dados diretamente na tabela sem recarregar a página.

## 📚 Aprendizados

Durante o desenvolvimento deste projeto foram explorados conceitos importantes como:

- Organização de rotas com Flask Blueprint
- Manipulação de banco de dados com SQLAlchemy
- Integração entre frontend e backend
- Uso de `fetch()` para comunicação AJAX
- Edição dinâmica de dados em tabelas HTML
